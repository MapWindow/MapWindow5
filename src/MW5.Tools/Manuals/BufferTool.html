Buffer by distance tool

  1. Select the layer to perform the buffer action.
  2. Optionally select if only the selected shapes should be used.
  3. Select the buffer distance.
  4. Select the units of the distance.
  5. Merge the results.
  6. The output layer name is generated based on the input layer, but can be modified.
  7. Check 'Overwrite' if you want to overwrite existing files. You'll be prompted otherwise if the output file already exists.
  8. Check 'Memory layer' if you don't want to save the file to disk, but make it only in memory.
  9. 'Add to map' is checked by default and will add the result to the map.
  10. Check if you want to run the process in the background, the progress can be monitored in the Task sidebar.

The number of segments are used in creating corners. More segments create rounder corners.
