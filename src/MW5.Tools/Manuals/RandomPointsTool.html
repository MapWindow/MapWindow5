Random points tool

If you need randomly generated point within a certain bounding box you can use this tool. Those points can, for example, be used to survey.
  1. Select the layer which bounding box will be used.
  2. Optionally select if only the selected shapes should be used for the bounding box.
  3. Select the number of points that should be created.
  4. The output layer name is generated based on the input layer, but can be modified.
  5. Check 'Overwrite' if you want to overwrite existing files. You'll be prompted otherwise if the output file already exists.
  6. Check 'Memory layer' if you don't want to save the file to disk, but make it only in memory.
  7. 'Add to map' is checked by default and will add the result to the map.
